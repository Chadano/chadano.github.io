Info

Hey, I'm Chadano (formally known as MrBeeBenson).

This is my little home on the internet where I like to show off things I have written or created! Here I will likely talk about myself, anything relating to open-source software and GNU/Linux ecosystems and anything relating to software (including software I create myself!)

I'm a software developer and information activist, I seek to promote accessible, reliable information about free and open source software in my blog posts and write-ups. I am 17 years old, and I am very passionate about computers and technology.

I first started learning website development in 2017 from school and fell in love with programming, from there I began to teach myself as much as I could. I was introduced to GNU/Linux and open source software in August 2020 and swapped from Windows 10 to Linux full-time in October 2020. I fell in love with open source software and the philosophy behind it.

I have recently converted back to Windows after giving the OS another try. I quickly fell in love with both Windows Terminal as well as Windows Subsystem for Linux 2 (WSL2), and without them I would not have been able to convert back to Windows. My love for Open Source Software still runs strong, and I will continue to create and contribute to FOSS software.

My projects

Rolling Rhino Remix - I am the founder and current Project Lead of Rolling Rhino Remix. Rolling Rhino Remix is a Linux distribution that converts Ubuntu into a Rolling Release distribution. Rolling Rhino Remix has many distribution-specific features and utilities specifically created for the distribution.

Chadachat - Chadachat is a self-hosted chatting application similar to AOL Instant Messenger. The chatting application is privacy-respecting and does not store any messages or user-data. Once a session has ended, all messages are gone.

skyWM - skyWM is a minimalistic window manager written in Rust. skyWM takes inspiration from the Unix philosophy of doing one thing and doing it well. skyWM has now become a part of the CatiumDE utilities and is maintained under one organisation.

Catium - Catium is a Desktop Environment with skyWM at its core. Catium combines elements of the traditional, conventional desktop layout, with a taskbar to easily switch between open applications within one workspace, with non-traditional elements such as a keyboard-focused tiling window management.

wsl-get - wsl-get is a tool which handles a collection of .deb packages that I (and contributors) use often when utilising WSL2. Rather than manually typing a wget command to download package files, this little tool automates the download and installation of packages.

This website - This is the current website you are reading. It is written in pure HTML and CSS with no JavaScript or additional libraries utilised.
